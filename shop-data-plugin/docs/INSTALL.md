# Chrome浏览器扩展安装指南

## 安装步骤

### 第一步：准备扩展文件

扩展文件已经准备好，位于：
```
/workspace/shop-data-plugin/extension/
```

### 第二步：打开Chrome扩展管理页面

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按回车键进入扩展管理页面

### 第三步：开启开发者模式

在扩展管理页面右上角，找到**"开发者模式"**开关，将其打开。

### 第四步：加载扩展

1. 点击左上角的**"加载已解压的扩展程序"**按钮
2. 在弹出的文件选择器中，选择扩展目录：
   ```
   /workspace/shop-data-plugin/extension/
   ```

### 第五步：确认安装成功

安装成功后，您会看到：
- 扩展列表中出现"店铺数据分析助手"
- 浏览器工具栏出现扩展图标

---

## 使用前配置

### 1. 启动后端服务

扩展需要后端API服务才能正常工作。在终端执行：

```bash
cd /workspace/shop-data-plugin/backend
python main.py
```

看到以下输出表示服务启动成功：
```
✓ 数据库初始化完成
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 2. 配置扩展API地址

1. 点击浏览器工具栏中的扩展图标
2. 如果看到登录界面，说明扩展已正常工作
3. 点击"设置"标签，确认API地址为：
   ```
   http://localhost:8000
   ```
4. 点击"测试连接"确认连接成功

### 3. 注册账号

1. 点击扩展图标
2. 点击"注册"标签
3. 输入用户名和密码
4. 点击注册完成账号创建

---

## 注意事项

⚠️ **重要提示**：

1. **后端服务必须运行**
   - 每次使用前，请确保后端服务已启动
   - 服务地址：http://localhost:8000

2. **开发者模式**
   - 由于是未打包的扩展，需要保持开发者模式开启
   - 每次打开Chrome可能会提示"禁用开发者模式扩展"，点击取消即可

3. **端口访问权限**
   - 扩展已配置 `http://localhost:*` 权限
   - 如需使用其他端口，需修改 `manifest.json`

## 故障排除

### 问题1：无法加载扩展

**解决方案**：
- 确保选择的是 `extension` 文件夹，而不是父目录
- 检查是否开启了开发者模式

### 问题2：点击图标没有反应

**解决方案**：
- 刷新扩展：在 `chrome://extensions/` 页面点击扩展卡片上的刷新按钮
- 检查控制台错误：右键扩展图标 → 检查弹出内容 → 查看Console

### 问题3：连接后端失败

**解决方案**：
- 确认后端服务已启动
- 检查API地址配置是否正确
- 尝试在浏览器直接访问 http://localhost:8000/health

### 问题4：拼多多页面抓取按钮不显示

**解决方案**：
- 刷新拼多多商家后台页面
- 确认已登录拼多多商家账号
- 检查content script是否正确加载
